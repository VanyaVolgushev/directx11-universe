The entire solar system featuring Gorbino's quest on the surface of the earth (Katamari Damacy). Made with C++/DirectX11.
Includes FPS, Isometric and orbital cameras, Phong lighting model, point lights, .obj parsing with textures and a simple ECS system to put it together.
