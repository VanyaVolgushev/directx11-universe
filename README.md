The entire solar system featuring Gorbino's quest on the surface of the earth (Katamari Damacy). Made with C++/DirectX11.

Includes:
1. FPS, Isometric and orbital cameras
2. Phong lighting model
3. Point lights with shadow maps
4. Assimp for .obj parsing with textures
5. A simple ECS system to put it together.
